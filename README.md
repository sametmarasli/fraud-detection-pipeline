# Fraud Detection 
Dataset: https://www.kaggle.com/datasets/ealaxi/paysim1

Used technologies:
- Orchestration: Airflow
- Data Streaming: Kafka
- ML Pipelines: scikit-learn
- ML Model: XGboost
- Monitoring: MLflow
- Docker
